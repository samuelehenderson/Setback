# 🃏 Setback (6-Point Pitch)

A card game for 2 or 4 players (teams) on separate phones.

## Rules
- **6 points per hand:** High, Low, Jack, Off Jack, Joker, Game
- **Trump order:** Joker → A → K → Q → J → Off Jack → 10 → 9...2
- **Bidding:** 3-6, highest bidder picks trump, dealer stuck at 3 if all pass
- **Play:** Must follow suit or play trump
- **Set back:** Bidder loses their bid if they don't make it
- **Win:** First to 21

## Setup

1. Open Terminal
2. Navigate to this folder:
   ```
   cd ~/Downloads/setback
   ```
3. Install (one time):
   ```
   npm install
   ```
4. Start:
   ```
   npm start
   ```
5. Open the URL shown on both phones (same WiFi!)
